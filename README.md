## gitでブランチを作る
ブランチとは枝のこと、開発において違う系統で行いたい場合はブランチ
かっこよく言うとブランチを切るという。

- 今のブランチを見る
```
git branch
```

- ブランチを作成する
```
git branch test_branch
```

- ブランチを移動
```
git checkout test_branch
```
